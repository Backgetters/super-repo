## 🎯 QUICK START

### Activate Super Environment
```bash
cd ~/clawd/super-repo
source activate.sh
```

### Start Core Services
```bash
# Start mission control dashboard
make mission-control-up

# Start deer-flow harness
make deer-flow-up

# Start all MCP servers
make mcp-up
```

### Deploy Agency Agents
```bash
# Deploy all agency agents
make agency-deploy

# Deploy specific agent
make agency-deploy AGENT=reddit-ninja
```

### Trading Operations
```bash
# Start trading bots
make trading-up

# Check positions
make trading-status
```
